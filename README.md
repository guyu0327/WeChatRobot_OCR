# WeChatRobot_OCR
基于OCR视觉识别的微信自动化工具

OCR使用的是网易有道的API <br>
云控制台链接：
https://ai.youdao.com/console/#/ <br>
文档链接：
https://ai.youdao.com/DOCSIRMA/html/ocr/api/tyocr/index.html

分析结果并自动回复使用了deepseek <br>
开放平台链接：
https://platform.deepseek.com/usage <br>
接口文档链接：
https://api-docs.deepseek.com/zh-cn/

由于每个人的电脑分辨率不同，且微信版本不同，
首次运行需要你先将image文件夹下的文件更新成自己的（在微信最大化的状态下截图替换原图），
其他则会自动更新

运行时，微信需要保持打开状态，且自动化操作会模拟人的行为去控制鼠标和键盘操作，
如果要长期运行，请使用一台空闲电脑或虚拟机
